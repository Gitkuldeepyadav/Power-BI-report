# Online sales dashboard 

"For this project, I wanted to analyze how a large retail business performs across the world. I used the Global Superstore 2016 dataset, which I downloaded from Kaggle."

## The Process 
### Data Cleaning: 
"My first step was to import the raw data into Power BI and clean it so that all the numbers and categories were accurate and ready for analysis.

### Building KPIs: 
"Next, I set up KPI metrics (Key Performance Indicators) at the top of my dashboard so that anyone looking at it could instantly see the most important numbers, like total sales and total profit."

### Creating Visuals: 
"Then, I used advanced features to build an interactive dashboard with specific charts to answer key business questions."

## The Details (Explaining the charts)

"To break down the data, I created a few specific visuals:"

"A Country-wise sales chart to see which countries were bringing in the most revenue."

"A ranking of the Top 5 customers and Top 5 products by city, which helps identify the most valuable buyers and best-selling items in specific locations."

"A Category-wise profit chart to understand exactly which types of products were making money and which ones were not."
